## Começando

* Dependência
  - PHP  
  - Laravel

* Para criar e construir seu banco de dados de acordo com o projeto.
  - db:User

* Para criar o banco de dados.
  - php artisan migrate
  
* Para executar o projeto.
  - Baixe e instale o laradock no projeto
