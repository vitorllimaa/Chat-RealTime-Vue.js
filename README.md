## Getting started

* Dependency
  - Ruby  
  - Rails

* To create and build your database accord of the project.
  - rake db:create db:migrate
  
* To create the previous files to use on the project, on the database.
  - rake db:seed
  
* To install the gems.
  - bundle install
  
* To run the project.
  - rails s
